Gistblog
========

Gistblog is a [simple blog engine based on github gists](https://gist.github.com/2821259).

To run:

1. install [node.js](https://github.com/joyent/node/wiki/Installation)

1. install [geddy](http://geddyjs.org/)

```cli
    npm install -g geddy
```    

1. start gistblog

```
    geddy
```
